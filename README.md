# AutoSignIn_Gamer
懶惰的新手工程師，經常忘記於巴哈姆特進行每日簽到，但又希望能夠拿到勇者徽章，所以只好寫支程式來輔助可憐的記憶力

# Feature
自動判斷目前是否尚未簽到，並進行簽到或是提示使用者已簽到。

# Use
1. 請先安裝腳本管理器(Tampermonkey or Violentmonkey)
2. 點選 AutoSignIn_Gamer.user.js > Raw 即可自動載入腳本
3. 台灣巴哈姆特https://www.gamer.com.tw/
